## Getting Started

1. Clone or download this repository
2. Open `Index.HTML` in your web browser
3. Or use a local server:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`

## Features in Detail

### Car Listings
- View all available cars in a grid layout
- Click "View Details" to see more information
- Filter by type (Rent/Sell), price range, and make

### Rental Form
- Select a car from available rentals
- Fill in your contact information
- Choose rental duration and pickup date

### Purchase Form
- Select a car from available sales
- Provide your contact and address information
- Submit a purchase request

### Sell Your Car
- List your car for rent or sale
- Add car details, price, mileage, and description
- Upload an image URL
